# Evolver Assets

`assets/gep/` mixes two different kinds of files:

- Curated repo-level Evolver configuration that should be versioned.
- Runtime state generated by local evolution runs that should stay untracked.

Tracked files:

- `genes.json`
- `scopes/competition-finish/genes.json`

Ignored runtime files include:

- `events.jsonl`
- `candidates.jsonl`
- `capsules.json`
- `failed_capsules.json`
- `genes.jsonl`
- any other scoped runtime state under `assets/gep/scopes/`

If MoonBitMark's preferred Evolver validation or judge-path strategy changes,
update both tracked gene files so root scope and `competition-finish` stay
consistent.
